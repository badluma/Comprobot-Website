# Comprobot Website

Marketing site for [Comprobot](https://github.com/badluma/Comprobot) — a highly
customizable, self-hostable open-source Discord bot.

Built with [Bun](https://bun.com) and HTML imports. No framework.

## Develop

Install dependencies:

```bash
bun install
```

Run with hot reload:

```bash
bun run index.ts
```

Then open the printed `localhost` URL.
