Comprobot

A highly customizable, self-hostable open-source Discord bot designed to be truly yours. Customize everything, bring your own AI models and be in control using the dashboard, all without writing any code.

install 
$ pipx install comprobot
$ comprobot onboard
Get started View on GitHub

A dashboard, not a config file

Run the web dashboard alongside the bot and manage everything from the browser. No need to manually edit configuration files.

  • Self-hosters
    You host the bot on your own server. You are in full control, no data is stored externally.
  • Tinkerers
    You can customize everything about the bot, with or without touching the code. The choice is yours.
  • Community servers
    Comprobot is made to adapt to your community's needs. That can be simple moderation, games or AI assistance. You can activate whatever you need.
  • Non-coders
    The all new dashboard makes it easy for everyone to have full control over your bot without writing any code or having to use the terminal the entire time.
dashboard 
$ bunx comprobot-dashboard

Opens at localhost:7626. Requires Comprobot installed and onboarded.

Built to be customized

Customizability

Comprobot is designed to be truly yours. You can customize basically everything, from the output format to the error messages to the AI system prompt, everything is adaptable.

Bring your own AI or turn it off

Most companies just force AI down your throat without letting you turn it off. In Comprobot, you can disable AI entirely by toggling one switch. You can also bring your own AI model.

Self-hosted

MIT-licensed and yours to run. No accounts, no telemetry, no data collection or anything else. Everything is open source and yours to modify.

Installs anywhere

Comprobot is available on basically any modern package manager, including pipx, APT, AUR, Homebrew, Winget, Scoop or Docker.
